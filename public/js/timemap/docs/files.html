Core functions for the timemap.js library. Timemap.js is intended to sync a SIMILE Timeline with a Google Map. Dependencies: Google Maps API v2, SIMILE Timeline v1.2 - 2.3.1 Thanks to Jorn Clausen (http://www.oe-files.de) for initial concept and code.

export.js

TimeMap Export Functions

Functions in this file allow TimeMap, TimeMapDataset, and TimeMapItem objects to be serialized to a JSON string suitable for loading back into TimeMap.init(). This allows for a range of server-side options for data persistence and management.

Depends on:

  • json2: lib/json2.pack.js
